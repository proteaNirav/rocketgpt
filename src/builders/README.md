# Mishti AI Builders

This directory contains the initial builder scaffolding for Mishti AI.

Builders are split into:
- `internal/` for Rust-oriented Internal Builders
- `external/` for TypeScript-oriented External Builders
- `shared/` for common contracts

Key rules:
- registration is mandatory
- no builder executes without registration
- wrapper mediation is mandatory
- current files are architecture-aligned skeletons, not production-complete logic
