# Mishti AI Builder Guardrail Profiles

## Purpose
This document defines the initial guardrail posture for Internal and External Builders.

## Internal Builder Profile
- minimum necessary guardrails
- mandatory registration lookup
- mandatory wrapper mediation
- trusted execution posture
- bounded permissions
- anomaly reporting

## External Builder Profile
- maximum guardrails
- mandatory registration lookup
- mandatory wrapper mediation
- stronger validation and sandboxing
- stricter policy checks
- stronger side-effect control

## Constraint
These profiles are architecture-aligned skeleton governance guidance only. They do not replace the existing platform guardrail backbone.
